# Interstice — App Directory

This is the Next.js application directory for Interstice. See the [root README](../README.md) for full project documentation, architecture, and setup instructions.

## Quick Start

```bash
npm install
npx convex dev          # Start Convex backend (keep running)
npm run seed            # Seed agent registry
npm run dev             # Start Next.js dev server
npm run heartbeat       # Start agent heartbeat scheduler (separate terminal)
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.
